title: 'Aside: one assumption'
---

**One fact taken on faith.** The argument needs that the arrow type $\mathsf{arr}\,A$ of a Segal type is *itself* Segal. This is a theorem of Riehl and Shulman, but its proof goes through the closure of Segal types under extension types, which rests on extension extensionality — machinery an order of magnitude larger than the reparametrisations this module is about. So we postulate it (`is-segal-arr`) and keep the focus on the geometry. It is the only assumption added here; everything else you build by hand.
