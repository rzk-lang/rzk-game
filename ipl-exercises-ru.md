# Упражнения по ИИВ (Telegram-сообщения)

Десять упражнений на построение дерева вывода в интуиционистском исчислении
высказываний (ИИВ) либо обоснование невыводимости. Сложность нарастает.
Каждый блок ниже — отдельное сообщение для Telegram (разделители `———`).

———

🧩 **Логические упражнения: интуиционистская логика высказываний**

Несколько задач на натуральный вывод (естественную дедукцию) в **ИИВ**.

Для каждой формулы нужно либо:
• построить **дерево вывода** в натуральном выводе (из пустого контекста), либо
• обосновать, что формула **невыводима** в ИИВ — например, предъявить
  контрмодель Крипке.

Обозначения: `∧` (и), `∨` (или), `→` (влечёт), `¬` (не), `⊥` (ложь).
Напомним: `¬A` — это сокращение для `A → ⊥`.

Правила: введение/устранение для `∧`, `∨`, `→`, а также `⊥`-устранение
(`ex falso`). Закон исключённого третьего и снятие двойного отрицания
**не** входят в число правил.

Задачи помечены звёздами по сложности: ⭐ — разминка, ⭐⭐⭐⭐⭐ — челлендж.

———

**Задача 1.** ⭐

Постройте дерево вывода или покажите невыводимость:

`(A → (B ∧ C)) → ((A → B) ∧ (A → C))`

Подсказка: это «обратная сторона» уже знакомой вам
`((A → B) ∧ (A → C)) → (A → (B ∧ C))`.

———

**Задача 2.** ⭐⭐

`((A ∧ B) → C) → (A → (B → C))`

Подсказка: каррирование. Из двух гипотез `A` и `B` соберите `A ∧ B`.

———

**Задача 3.** ⭐⭐

`(A ∨ B) → (B ∨ A)`

Подсказка: понадобится устранение дизъюнкции (разбор случаев).

———

**Задача 4.** ⭐⭐

`(A → (B → C)) → ((A → B) → (A → C))`

Подсказка: это «комбинатор S». Аккуратно следите за тем, какие гипотезы
ещё не сняты.

———

**Задача 5.** ⭐⭐⭐

`((A → C) ∧ (B → C)) → ((A ∨ B) → C)`

———

**Задача 6.** ⭐⭐⭐

`A → ¬¬A`

Напоминаю: `¬¬A` — это `(A → ⊥) → ⊥`.

———

**Задача 7.** ⭐⭐⭐⭐

`¬¬A → A`

Прежде чем строить дерево, подумайте: а выводимо ли это вообще в ИИВ?

———

**Задача 8.** ⭐⭐⭐⭐

`(A → B) ∨ (B → A)`

Классически — тавтология. А в ИИВ?

———

**Задача 9.** ⭐⭐⭐⭐

`((A → B) → A) → A`

Это закон Пирса. Классически он истинен — но нас интересует ИИВ.

———

**Задача 10.** ⭐⭐⭐⭐⭐

`¬¬(A ∨ ¬A)`

Сравните с законом исключённого третьего `A ∨ ¬A`. Совпадает ли ответ?

———

🔒 **Ключ для преподавателя (спойлер!)**

1. **Выводима.** Снимите `A → (B∧C)` и `A`. Применение даёт `B∧C`;
   устранением `∧` получаем `B`, откуда `A → B`; аналогично `A → C`.
   Введением `∧` собираем `(A→B) ∧ (A→C)`. (Обратная формула,
   которую студенты уже умеют, — другое направление этой же
   эквивалентности.)

2. **Выводима** (каррирование). Снимите `(A∧B)→C`, затем `A`, затем `B`.
   Введением `∧` соберите `A∧B`, примените `(A∧B)→C` — получите `C`.
   Тремя введениями `→` снимите гипотезы.

3. **Выводима.** Разбор случаев по `A ∨ B`: в ветке с `A` вводим правую
   сторону `B ∨ A`, в ветке с `B` — левую.

4. **Выводима.** Снимаются три гипотезы: `A→(B→C)`, `A→B`, `A`.
   Из `A` получаем `B` (через `A→B`) и `B→C` (через `A→(B→C)`),
   откуда `C`. Это аксиома S.

5. **Выводима.** Снимите `(A→C)∧(B→C)` и `A∨B`. Разбор случаев по `A∨B`:
   в ветке `A` применяем `A→C`, в ветке `B` — `B→C`; обе дают `C`.

6. **Выводима.** Снимите `A`, затем `A→⊥`; устранение `→` даёт `⊥`;
   введение `→` даёт `¬¬A`. Заметьте: обратное `¬¬A → A` (задача 7)
   уже не проходит.

7. **Невыводима.** Контрмодель Крипке: два мира `w₀ ≤ w₁`, переменная
   `A` ложна в `w₀` и истинна в `w₁`. Тогда `¬¬A` истинно в `w₀`
   (ни в одном достижимом мире `¬A` не выполнено), а `A` в `w₀` ложно.
   Значит `¬¬A → A` опровергается в `w₀`.

8. **Невыводима.** Нужна ветвящаяся модель: корень `w₀` с двумя
   несравнимыми наследниками `w₁` и `w₂`. В `w₁` истинно `A`, ложно `B`;
   в `w₂` истинно `B`, ложно `A`. Тогда `A→B` ложно в `w₀`
   (свидетель — `w₁`), и `B→A` ложно в `w₀` (свидетель — `w₂`),
   поэтому дизъюнкция `(A→B) ∨ (B→A)` в `w₀` ложна.

9. **Невыводима** (закон Пирса). Контрмодель: миры `w₀ ≤ w₁`,
   `A` ложно в `w₀`, истинно в `w₁`; `B` ложно везде. Тогда `A→B`
   ложно в обоих мирах, поэтому `(A→B)→A` истинно в `w₀` вакуумно,
   но `A` в `w₀` ложно — вся формула опровергается.

10. **Выводима!** В этом и соль: само `A ∨ ¬A` невыводимо,
    а его двойное отрицание — выводимо. Снимите гипотезу `h : ¬(A∨¬A)`.
    Внутри: снимите `A`, введением `∨` получите `A∨¬A`, примените `h` —
    получите `⊥`; сняв `A`, получите `¬A`. Снова введением `∨`
    получите `A∨¬A`, примените `h` — снова `⊥`. Сняв `h`, получите
    `¬¬(A ∨ ¬A)`.
